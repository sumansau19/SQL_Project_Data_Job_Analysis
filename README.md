# SQL Data Job Analysis Project

## Project Overview
This project analyzes data job postings to identify:
- Highest paying skills
- Most in-demand roles
- Remote vs onsite jobs
- Salary trends

## Tools Used
- SQL
- PostgreSQL
- VS Code

## Key Insights
- Python and SQL are highest in demand
- Remote jobs offer competitive salaries
- Data Engineers have highest salary range
